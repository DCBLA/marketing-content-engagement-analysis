# Data

The raw dataset is not included in this portfolio version.

To reproduce the analysis, place the original file here:

```text
data/raw/Global_media_data.csv
```

The analysis script expects an article-level dataset with 39,644 rows and 61 columns. The target variable is `Engagement`, defined as total social shares.

The public portfolio version excludes the dataset because it was supplied in an academic setting and should not be republished without confirming usage rights.
